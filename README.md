## 🕹️ 遊戲說明：數字生存挑戰 (Number Survival)

這不僅僅是一個「猜數字」遊戲，這是一場關於**資源管理**與**直覺**的對抗。

### 🌟 核心特色
* **體力系統**：你不是有無窮無盡的機會，每次猜錯都會消耗 `❤️`。
* **防呆機制**：輸入非數字字符不會崩潰，系統會溫柔地提醒你。
* **動態回饋**：根據你的輸入提供即時的範圍提示（太大/太小）。

### 🛠️ 技術規格
* **語言**: Python 3
* **核心模組**: `random` (隨機數生成)
* **邏輯架構**: 
    * `while` 迴圈控制遊戲生命週期。
    * `try-except` 捕捉異常輸入。
